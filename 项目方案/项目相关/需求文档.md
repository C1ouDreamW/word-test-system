# 专业英语单词测试系统 — 需求文档

> **选题**：4．专业英语单词测试（2人，A级）

---

## 一、任务书原话

> 利用已有的专业英语单词库，构建一个可以实时出题的英文单词考试系统。  
> 主要功能：发起一次考试（考试有起止时间），动态生成考试页面（每一次考试所有考生题目一致，但是对每个考生随机打乱题序）、自动判卷、生成考试统计（以页面或文件的形式）。字符云功能：可根据学生每周回答的情况，动态生成字符云。  
> 可改进的地方：支持选择题。

---

## 二、功能列表

| # | 功能 | 说明 |
|---|------|------|
| ① | **单词库** | 管理员可增删改查英语单词，支持预置初始数据 |
| ② | **限时考试** | 管理员发起考试（起止时间、题目数）；学生答题时系统随机打乱题序（题目一致但每人顺序不同），倒计时自动提交 |
| ③ | **自动判卷 + 统计** | 提交后自动判卷出分；管理员查看成绩表（按分降序），含平均分、最高分、各题正确率 |
| ④ | **字符云** | 根据学生每周答题情况生成字符云——常错词突出显示，学生和管理员均可查看 |
| 🔧 | **选择题**（改进） | 在英译中、中译英之外，增加选择题（给英文，4 个中文选项选 1） |

三种题型：

| 题型 | 类别 |
|------|:--:|
| 英译中（看英文写中文） | 基本 |
| 中译英（看中文写英文） | 基本 |
| 选择题（看英文选正确中文，4 选 1） | 改进 |

## 三、用户与登录

- 不设注册，数据库预置账号（学生 ×3 + 管理员 ×1）
- 登录页：选角色 → 输用户名 → 进入首页

---

## 四、页面（8 页）

```
/login.html                    登录

学生：
/student/dashboard.html        首页（考试列表 + 历史成绩）
/student/exam.html             答题页
/student/result.html           交卷结果（分数 + 错题）

管理员：
/admin/words.html              单词库管理
/admin/exams.html              考试管理（创建 + 列表）
/admin/stats.html              成绩统计
/admin/wordcloud.html          字符云
```

---

## 五、数据库（5 张表）

```
users           — id, username, role(admin/student)
words           — id, english, chinese, category
exams           — id, title, start_time, end_time, question_count
questions       — id, exam_id, user_id, word_id, question_type, question_text, answer, sort_order
answers         — id, question_id, user_id, user_answer, is_correct
```
