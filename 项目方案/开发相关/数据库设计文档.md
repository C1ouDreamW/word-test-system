# 数据库设计文档

> 数据库：MySQL 8.0  
> 数据库名：`word_test`  
> 字符集：utf8mb4

---

## 一、表结构

### 1. users — 用户表

| 字段 | 类型 | 长度 | 非空 | 默认值 | 说明 |
|------|------|:--:|:--:|------|------|
| id | INT | — | ✅ | AUTO_INCREMENT | 主键 |
| username | VARCHAR | 50 | ✅ | | 用户名 |
| role | VARCHAR | 20 | ✅ | | admin / student |

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    role VARCHAR(20) NOT NULL COMMENT 'admin或student'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

---

### 2. words — 单词表

| 字段 | 类型 | 长度 | 非空 | 说明 |
|------|------|:--:|:--:|------|
| id | INT | — | ✅ | 主键 |
| english | VARCHAR | 100 | ✅ | 英文单词 |
| chinese | VARCHAR | 200 | ✅ | 中文释义 |
| category | VARCHAR | 100 | | 分类/章节 |

```sql
CREATE TABLE words (
    id INT AUTO_INCREMENT PRIMARY KEY,
    english VARCHAR(100) NOT NULL,
    chinese VARCHAR(200) NOT NULL,
    category VARCHAR(100) COMMENT '分类，如：网络基础、路由协议'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

---

### 3. exams — 考试表

| 字段 | 类型 | 长度 | 非空 | 说明 |
|------|------|:--:|:--:|------|
| id | INT | — | ✅ | 主键 |
| title | VARCHAR | 200 | ✅ | 考试名称 |
| start_time | DATETIME | — | ✅ | 开始时间 |
| end_time | DATETIME | — | ✅ | 结束时间 |
| question_count | INT | — | ✅ | 题目数量 |

```sql
CREATE TABLE exams (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(200) NOT NULL,
    start_time DATETIME NOT NULL,
    end_time DATETIME NOT NULL,
    question_count INT NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

---

### 4. questions — 题目表

> 学生首次获取考卷时生成，持久化以保证同一学生的考卷不变。

| 字段 | 类型 | 长度 | 非空 | 说明 |
|------|------|:--:|:--:|------|
| id | INT | — | ✅ | 主键 |
| exam_id | INT | — | ✅ | 关联 exams.id |
| user_id | INT | — | ✅ | 关联 users.id |
| word_id | INT | — | ✅ | 关联 words.id |
| question_type | VARCHAR | 20 | ✅ | EN_TO_CN / CN_TO_EN / CHOICE |
| question_text | VARCHAR | 200 | ✅ | 题干 |
| correct_answer | VARCHAR | 200 | ✅ | 正确答案 |
| options | VARCHAR | 500 | | 选择题选项，JSON 数组，非选择题为 NULL |
| sort_order | INT | — | ✅ | 排序号（体现随机题序） |

```sql
CREATE TABLE questions (
    id INT AUTO_INCREMENT PRIMARY KEY,
    exam_id INT NOT NULL,
    user_id INT NOT NULL,
    word_id INT NOT NULL,
    question_type VARCHAR(20) NOT NULL COMMENT 'EN_TO_CN / CN_TO_EN / CHOICE',
    question_text VARCHAR(200) NOT NULL,
    correct_answer VARCHAR(200) NOT NULL,
    options VARCHAR(500) COMMENT 'JSON数组，选择题用',
    sort_order INT NOT NULL,
    FOREIGN KEY (exam_id) REFERENCES exams(id),
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (word_id) REFERENCES words(id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

---

### 5. answers — 答案表

| 字段 | 类型 | 长度 | 非空 | 说明 |
|------|------|:--:|:--:|------|
| id | INT | — | ✅ | 主键 |
| question_id | INT | — | ✅ | 关联 questions.id |
| user_id | INT | — | ✅ | 关联 users.id |
| user_answer | VARCHAR | 200 | | 学生提交的答案 |
| is_correct | TINYINT | — | ✅ | 0 错 / 1 对 |

```sql
CREATE TABLE answers (
    id INT AUTO_INCREMENT PRIMARY KEY,
    question_id INT NOT NULL,
    user_id INT NOT NULL,
    user_answer VARCHAR(200),
    is_correct TINYINT NOT NULL DEFAULT 0 COMMENT '0错/1对',
    FOREIGN KEY (question_id) REFERENCES questions(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

---

## 二、ER 关系

```
users ──┬── questions ──── words
        │       │
        │   answers
        │
exams ──┼── questions
```
