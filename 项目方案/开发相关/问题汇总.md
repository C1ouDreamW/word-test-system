## 发现的Bug汇总

- [ ] 表示未解决的bug；
- [x] 表示已解决的bug。

---

- [ ] 同category的单词，可能会出现重复的情况，导致在做题时，出现重复的题目。
  - 解决方案：在查询单词时，使用`DISTINCT`关键字，去重。
- [x] `POST /api/words` 新增单词返回的 `id` 为 `null`
  - 原因：`WordDaoImpl.insert()` 使用 `jdbc.update()` 执行 INSERT，未获取数据库自增主键
  - 修复：改用 `PreparedStatement` + `GeneratedKeyHolder` 捕获自增 Key 并回填到实体
- [x] `POST /api/exams` 创建考试返回的 `id` 为 `null`
  - 原因：`ExamDaoImpl.insert()` 同样使用 `jdbc.update()` 执行 INSERT，未获取自增主键
  - 修复：改用 `PreparedStatement` + `GeneratedKeyHolder` 捕获自增 Key 并回填到实体
