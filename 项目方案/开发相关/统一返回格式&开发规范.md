# 统一返回格式 & 开发规范

---

## 一、统一返回体：ApiResult

所有 Controller 返回同一结构，前端统一处理。

```java
public class ApiResult<T> {
    private int code;       // 200 成功，其他为错误码
    private String message; // 提示信息，"ok" 或错误描述
    private T data;         // 业务数据

    // 成功
    public static <T> ApiResult<T> ok(T data) {
        return new ApiResult<>(200, "ok", data);
    }

    // 失败
    public static <T> ApiResult<T> fail(String message) {
        return new ApiResult<>(400, message, null);
    }
}
```

**JSON 示例：**
```json
// 成功
{ "code": 200, "message": "ok", "data": { "id": 1, "username": "zhangsan" } }

// 失败
{ "code": 400, "message": "单词不存在", "data": null }
```

## 二、Controller 写法规范

```java
@RestController
@RequestMapping("/api/words")
public class WordController {

    @Autowired
    private WordService wordService;

    @GetMapping
    public ApiResult<List<Word>> list() {
        return ApiResult.ok(wordService.findAll());
    }

    @PostMapping
    public ApiResult<Word> add(@RequestBody Word word) {
        return ApiResult.ok(wordService.add(word));
    }
}
```

- 类上 `@RestController`（所有方法自动返回 JSON）
- 路径统一 `/api/` 前缀
- 返回值**永远用 ApiResult 包裹**，不要直接返回裸对象

## 三、DAO 写法规范

任务书要求 DAO 接口 + 实现类模式：

```java
// 接口
public interface WordDao {
    List<Word> findAll();
    Word findById(int id);
    int insert(Word word);
    int update(Word word);
    int delete(int id);
}

// 实现类
@Repository
public class WordDaoImpl implements WordDao {

    @Autowired
    private JdbcTemplate jdbc;

    @Override
    public List<Word> findAll() {
        String sql = "SELECT id, english, chinese, category FROM words";
        return jdbc.query(sql, new BeanPropertyRowMapper<>(Word.class));
    }

    @Override
    public int insert(Word word) {
        String sql = "INSERT INTO words (english, chinese, category) VALUES (?, ?, ?)";
        return jdbc.update(sql, word.getEnglish(), word.getChinese(), word.getCategory());
    }
}
```

## 五、命名约定

| 类型 | 规则 | 示例 |
|------|------|------|
| 类名 | 大驼峰 | `ExamController` |
| 方法/变量 | 小驼峰 | `findAll`, `examId` |
| 包名 | 全小写 | `com.scut.wordtest` |
| 表名 | 全小写复数 | `users`, `exams` |
| 列名 | 全小写下划线 | `question_count` |
| URL 路径 | 全小写复数 | `/api/words`, `/api/exams` |
