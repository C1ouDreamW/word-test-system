## 二、环境配置步骤

### 2.1 JDK 17

```bash
# 安装后验证
java -version
# → openjdk version "17.0.x" ...

# 确认 JAVA_HOME 环境变量已设置
echo $JAVA_HOME
# → C:\Program Files\...
```

### 2.2 Maven 3.6.3+

```bash
# 验证
mvn -version
# → Apache Maven 3.6.3+（Spring Boot 3.x 要求 ≥3.6.3）

# 配置阿里云镜像（加速依赖下载）
# 编辑 ~/.m2/settings.xml，添加：
```

```xml
<mirrors>
    <mirror>
        <id>aliyun</id>
        <mirrorOf>central</mirrorOf>
        <name>阿里云 Maven 镜像</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </mirror>
</mirrors>
```

### 2.3 MySQL 8.0

```sql
-- 登录 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE word_test DEFAULT CHARSET utf8mb4;

-- 执行建表脚本
-- 或直接复制数据库设计文档中的 SQL
```

### 2.4 Git 配置

```bash
git config --global user.name "你的姓名"
git config --global user.email "你的邮箱"

# 克隆项目（初始化后）
git clone <仓库地址>
cd word-test-system
```

## 三、项目启动

### 3.1 IDEA 打开项目

```
File → Open → 选择项目根目录（pom.xml 所在目录）
IDEA 自动识别 Maven 项目，下载依赖（首次需要几分钟）
```

### 3.2 配置 application.properties

```properties
# src/main/resources/application.properties

server.port=8080

spring.datasource.url=jdbc:mysql://localhost:3306/word_test?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf-8
spring.datasource.username=root
spring.datasource.password=你的密码
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
```

### 3.3 启动

```
运行 WordTestApplication.java 的 main() 方法
控制台显示：Started WordTestApplication in x seconds
浏览器访问：http://localhost:8080
```

### 3.4 打包

```bash
mvn clean package -DskipTests
# jar 文件在 target/ 目录下

# 运行 jar
java -jar target/word-test-0.0.1-SNAPSHOT.jar
```
