# 专业英语单词测试系统

> **选题**：4．专业英语单词测试 ｜ **等级**：A ｜ **人数**：2  
> **技术栈**：Spring Boot 2.7 + REST API + Bootstrap 5 + MySQL 8.0

---

## 一、项目简介

面向「网络工程导论」课程的专业英语单词限时考试系统。支持管理员发布考试、学生在线答题（题目随机排序）、自动判卷、成绩统计，以及根据答题情况生成错词字符云。

---

## 二、快速开始

### 环境要求

- JDK 8
- Maven 3.6+
- MySQL 8.0

### 启动步骤

```bash
# 1. 初始化数据库
mysql -u root -p < sql/init.sql

# 2. 修改数据库连接（如果密码不是 root）
# 编辑 src/main/resources/application.properties → spring.datasource.password

# 3. 启动项目
mvn spring-boot:run

# 4. 浏览器访问
# http://localhost:8080/login.html
```

## 三、开发流程

### 分支策略

```
main        ← 稳定版本，每天合并
├── dev-a   ← A 开发分支（组长）
└── dev-b   ← B 开发分支
```

### 每日流程

```bash
# 上午 → 拉最新代码
git checkout main && git pull
git checkout dev-a && git merge main

# 开发中 → 按模块提交
git commit -m "feat(考试): 完成组卷算法"

# 收工前 → 合并到 main
git checkout main && git pull && git merge dev-a && git push
```

### Commit 格式

```
type(scope): 说明

feat  → 新功能
fix   → 修 bug
docs  → 文档
chore → 构建/配置
```

详细规范见：[分支管理与提交规范](项目方案/开发相关/分支管理与提交规范.md)

### 开发顺序

| 阶段 | 天数 | 内容 |
|------|:--:|------|
| 后端 | D1–D4 | 数据库建表 → DAO → Service → Controller |
| 前端 | D5–D7 | 页面搭建 → 接口联调 → 选择题 → 字符云 |
| 收尾 | D8 | 规约扫描 → 修 bug → 报告 + PPT |

## 四、功能清单

### 必做功能（任务书要求）

| # | 功能 | 说明 | 状态 |
|---|------|------|:--:|
| ① | 单词库管理 | 管理员增删改查，预置 40 个网络工程词汇 | ⬜ |
| ② | 限时考试 | 管理员发起考试（起止时间、题目数）；学生答题时系统随机打乱题序，倒计时自动提交 | ⬜ |
| ③ | 自动判卷 + 统计 | 提交后自动判卷出分；管理员查看成绩表，含平均分、最高分、各题正确率 | ⬜ |
| ④ | 字符云 | 根据学生答题情况生成错词字符云，错得越多显示越大 | ⬜ |
| 🔧 | 选择题（改进项） | 在英译中、中译英之外，增加选择题（4 选 1，随机干扰项） | ⬜ |

### 用户登录

| 功能 | 说明 | 状态 |
|------|------|:--:|
| 登录 | 预置账号，角色区分（管理员/学生），不设注册 | ⬜ |
